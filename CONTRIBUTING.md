# Contributing to Trios: 734

First off, thank you for considering contributing to this project!

## How to contribute

### Reporting bugs
- Use the **Issues** tab
- Include: game version, steps to reproduce, expected vs actual behavior
- Or report on our [Discord](https://discord.gg/GEyVXwjMuV)

### Suggesting features
- Open an issue with label `enhancement`
- Describe the feature and why it fits the game's dark sci-fi theme

### Pull Requests
1. Fork the repo
2. Create a branch (`git checkout -b feature/amazing-feature`)
3. Commit your changes
4. Push to the branch
5. Open a Pull Request

## Code of Conduct

Be respectful. This is a community project.

## Questions?

Join our [Discord](https://discord.gg/GEyVXwjMuV)
